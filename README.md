# Splitora
Splitora is a smart, hassle-free expense splitter that makes sharing costs simple and fair. It helps groups track expenses, split bills, analyze charts, and settle debts effortlessly. Whether you're traveling, renting, or organizing events, Splitora ensures every expense finds its perfect balance. With multi-currency support, clear visual charts, and a sleek, intuitive design, it brings clarity, fairness, and peace of mind to group finances where fairness meets finance.
Key Features
Currency selection for global usage

Percentage-based expense distribution

Categorized expense tracking (e.g., Food, Travel, Rent)

Regular budget alerts and reminders

Set monthly or event-specific budget limits

Collaborate with UPI apps for seamless settlements

View individual and group account balances

Visual heatmap of transactions to analyze spending patterns
